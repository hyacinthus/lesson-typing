# i18n Subpath Routing Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Enable Google to index each language version separately via URL-based language paths (`/en/`, `/zh/`, etc.) using build-time template injection.

**Architecture:** Build script generates per-language `index.html` files with localized `<head>` meta tags after `vite build`. React Router maps `/:lang/` to the existing HomePage. Root `/` redirects to the detected language. The app remains a pure SPA.

**Tech Stack:** React Router v7, Node.js build script (CJS), i18next, Caddy

### Task 2: Create the build-time template injection script

This script runs after `vite build`. It reads `dist/index.html` as a template, generates per-language copies with localized `<head>` tags, replaces root `index.html` with a redirect page, and generates `sitemap.xml`.

**Files:**
- Create: `scripts/generate-i18n-pages.cjs`
- Modify: `package.json:9` (build script)
- Delete: `public/sitemap.xml` (now generated by script)

**Step 1: Write the build script**

Create `scripts/generate-i18n-pages.cjs`:

```js
const fs = require('fs');
const path = require('path');

const meta = require('../src/i18n-meta.json');
const DIST = path.join(__dirname, '..', 'dist');
const BASE_URL = meta.baseUrl;
const LANGS = Object.keys(meta.languages);

// Read the vite-built index.html as template
const template = fs.readFileSync(path.join(DIST, 'index.html'), 'utf-8');

// Build hreflang link tags (same for all pages)
const hreflangTags = LANGS.map(
  (l) => `    <link rel="alternate" hreflang="${l}" href="${BASE_URL}/${l}/" />`
).concat([`    <link rel="alternate" hreflang="x-default" href="${BASE_URL}/" />`])
  .join('\n');

// Generate per-language index.html
for (const lang of LANGS) {
  const { title, description, ogDescription } = meta.languages[lang];

  let html = template;

  // <html lang="...">
  html = html.replace(/<html lang="[^"]*"/, `<html lang="${lang}"`);

  // <title>
  html = html.replace(/<title>[^<]*<\/title>/, `<title>${title}</title>`);

  // <meta name="description">
  html = html.replace(
    /<meta name="description" content="[^"]*"/,
    `<meta name="description" content="${description}"`
  );

  // og:title, og:description, og:url
  html = html.replace(
    /<meta property="og:title" content="[^"]*"/,
    `<meta property="og:title" content="${title}"`
  );
  html = html.replace(
    /<meta property="og:description" content="[^"]*"/,
    `<meta property="og:description" content="${ogDescription}"`
  );
  html = html.replace(
    /<meta property="og:url" content="[^"]*"/,
    `<meta property="og:url" content="${BASE_URL}/${lang}/"`
  );

  // twitter:title, twitter:description
  html = html.replace(
    /<meta name="twitter:title" content="[^"]*"/,
    `<meta name="twitter:title" content="${title}"`
  );
  html = html.replace(
    /<meta name="twitter:description" content="[^"]*"/,
    `<meta name="twitter:description" content="${ogDescription}"`
  );

  // canonical
  html = html.replace(
    /<link rel="canonical" href="[^"]*"/,
    `<link rel="canonical" href="${BASE_URL}/${lang}/"`
  );

  // Structured data: name and description
  html = html.replace(
    /"name": "Lesson Typing"/,
    `"name": "${title.replace(/"/g, '\\"')}"`
  );
  html = html.replace(
    /"url": "https:\/\/lessontyping.com"/,
    `"url": "${BASE_URL}/${lang}/"`
  );
  html = html.replace(
    /"description": "Free typing practice[^"]*"/,
    `"description": "${description.replace(/"/g, '\\"')}"`
  );

  // Inject hreflang tags before </head>
  html = html.replace('</head>', `\n${hreflangTags}\n  </head>`);

  // Write to dist/{lang}/index.html
  const langDir = path.join(DIST, lang);
  fs.mkdirSync(langDir, { recursive: true });
  fs.writeFileSync(path.join(langDir, 'index.html'), html);
  console.log(`  Generated ${lang}/index.html`);
}

// Replace root index.html with redirect page
const rootHtml = `<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Lesson Typing</title>
    <meta name="description" content="Free typing practice for elementary school students in 8 languages." />
${hreflangTags}
    <link rel="icon" type="image/png" href="/favicon.png" />
    <script>
      (function() {
        var LANGS = ${JSON.stringify(LANGS)};
        var stored = null;
        try { stored = localStorage.getItem('lesson-typing-language'); } catch(e) {}
        var lang = stored || (navigator.language || '').split('-')[0];
        if (LANGS.indexOf(lang) === -1) lang = 'en';
        location.replace('/' + lang + '/');
      })();
    </script>
  </head>
  <body>
    <noscript><meta http-equiv="refresh" content="0;url=/en/"></noscript>
  </body>
</html>
`;
fs.writeFileSync(path.join(DIST, 'index.html'), rootHtml);
console.log('  Generated root index.html (redirect)');

// Generate sitemap.xml
const hreflangXml = LANGS.map(
  (l) => `      <xhtml:link rel="alternate" hreflang="${l}" href="${BASE_URL}/${l}/" />`
).concat([`      <xhtml:link rel="alternate" hreflang="x-default" href="${BASE_URL}/" />`])
  .join('\n');

const sitemapUrls = LANGS.map((l) => `  <url>
    <loc>${BASE_URL}/${l}/</loc>
    <changefreq>weekly</changefreq>
    <priority>0.9</priority>
${hreflangXml}
  </url>`).join('\n');

const sitemap = `<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:xhtml="http://www.w3.org/1999/xhtml">
  <url>
    <loc>${BASE_URL}/</loc>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
${hreflangXml}
  </url>
${sitemapUrls}
</urlset>
`;
fs.writeFileSync(path.join(DIST, 'sitemap.xml'), sitemap);
console.log('  Generated sitemap.xml');

console.log('i18n page generation complete.');
```

**Step 1: Update App.tsx with router config**

Replace the entire `src/App.tsx` with:

```tsx
import { useEffect } from 'react';
import { BrowserRouter, Routes, Route, Navigate, useParams, useNavigate, Outlet } from 'react-router-dom';
import { useTranslation } from 'react-i18next';
import { Toaster } from '@/components/ui/sonner';
import { LoginOverlay } from '@/components/ui/login-overlay';
import { HomePage } from './pages/HomePage';
import { useAuthStore } from './stores/authStore';

const SUPPORTED_LANGS = ['en', 'zh', 'es', 'ja', 'pt', 'fr', 'de', 'it'];
const LANG_STORAGE_KEY = 'lesson-typing-language';

function RootRedirect() {
  let lang: string | null = null;
  try {
    lang = localStorage.getItem(LANG_STORAGE_KEY);
  } catch { /* ignore */ }
  if (!lang || !SUPPORTED_LANGS.includes(lang)) {
    const browserLang = navigator.language?.split('-')[0];
    lang = SUPPORTED_LANGS.includes(browserLang) ? browserLang : 'en';
  }
  return <Navigate to={`/${lang}/`} replace />;
}

function LangLayout() {
  const { lang } = useParams<{ lang: string }>();
  const { i18n } = useTranslation();
  const navigate = useNavigate();

  // Validate lang param
  if (!lang || !SUPPORTED_LANGS.includes(lang)) {
    return <Navigate to="/en/" replace />;
  }

  // Sync i18n language with URL param
  useEffect(() => {
    if (i18n.language.split('-')[0] !== lang) {
      i18n.changeLanguage(lang);
    }
  }, [lang, i18n]);

  // Persist language choice
  useEffect(() => {
    try {
      localStorage.setItem(LANG_STORAGE_KEY, lang);
    } catch { /* ignore */ }
  }, [lang]);

  return <Outlet context={{ navigate, lang, supportedLangs: SUPPORTED_LANGS }} />;
}

function App() {
  const { initialize, isLoggingIn } = useAuthStore();

  useEffect(() => {
    initialize();
  }, [initialize]);

  return (
    <BrowserRouter>
      <Routes>
        <Route index element={<RootRedirect />} />
        <Route path=":lang/*" element={<LangLayout />}>
          <Route index element={<HomePage />} />
        </Route>
      </Routes>
      <LoginOverlay isVisible={isLoggingIn} />
      <Toaster />
    </BrowserRouter>
  );
}

export default App;
```

### Task 4: Update Caddy configuration

The Caddy `try_files` directive needs to check for subdirectory index files (e.g., `/zh/` → `/zh/index.html`) before falling back. Also add no-cache headers for language index files.

**Files:**
- Modify: `Caddyfile`

**Step 1: Update Caddyfile**

Replace the entire `Caddyfile` with:

```
:80 {
    root * /srv
    encode gzip zstd

    # SPA fallback: try exact path, then subdir index, then language index, then root
    @langPath path_regexp lang ^/([a-z]{2})/
    try_files {path} {path}/index.html /index.html

    # Security headers
    header Referrer-Policy strict-origin-when-cross-origin
    header X-Content-Type-Options nosniff
    header X-Frame-Options DENY

    # Cache immutable assets (JS/CSS with hash in filename)
    header /assets/* Cache-Control "public, max-age=31536000, immutable"

    # Don't cache HTML entry points (root + language index files)
    header /index.html Cache-Control "no-cache, no-store, must-revalidate"
    header /*/index.html Cache-Control "no-cache, no-store, must-revalidate"

    # Cache lesson data with revalidation
    header /lessons/* Cache-Control "public, max-age=3600, stale-while-revalidate=86400"

    file_server
}
```

Key change: `try_files {path} {path}/index.html /index.html` — this makes `/zh/` serve `zh/index.html`, and any client-side route like `/zh/some-deep-path` falls through to `/index.html` where the SPA router handles it.
